WeBots Grocery Shopper

Project Details

Technologies: WeBots, Python

Duration: 2 months

The Story

This project implements a robotic system in Webots capable of mapping, localization, computer vision, navigation, and manipulation to autonomously find and pick up colored cubes in a simulated grocery environment.

Challenges Conquered

  • Mapping & Localization: Using LiDAR, GPS, and odometry with transformation matrices.
  • Computer Vision: Blob detection with OpenCV to identify colored blocks.
  • Navigation: RRT-based planning for navigating between cube pickup points.
  • Manipulation: Inverse kinematics for arm control to pick up and place objects.
